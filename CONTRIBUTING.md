# Contributing to Slipstream Python SDK

First off, thanks for taking the time to contribute!

## Development Setup

1.  Install dependencies (e.g., `pip install -e .[dev]`).
2.  Run tests using `pytest`.
3.  Ensure code follows standard Python guidelines (PEP 8).

## Pull Request Process

1.  Fork the repo and create your branch from `main`.
2.  Add tests for any new functionality.
3.  Ensure the test suite passes.
4.  Update the documentation.
5.  Sign your commits (DCO).

## Code Style

We use `ruff` or `black` for formatting where applicable.
